# KinOS

KinOS is a local-first infrastructure for personal AI agents living inside human groups.

The first use case is the family: each member has a personal agent, private memory, governed shared memory, authorized capabilities, and a collective family agent.

The repository currently contains the foundational product contracts and the first architecture decision record.

- [Results Contract](docs/contracts/results-contract.md)
- [Invariants Contract](docs/contracts/invariants-contract.md)

## Core principle

KinOS is not a family chatbot.

It is an infrastructure of trust for personal agents operating inside a real human collective.

Models can change. Tools can change. Integrations can change. Interfaces can change.

Data sovereignty, privacy, consent, portability, and safety must not change.
