# KinOS — Glossary

## Agent

A digital representative acting for a person or Sphere within authorized boundaries.

## Audit Event

A minimal security-relevant fact recorded for traceability.

## Capability

A governed abstract action that an agent can request, such as `memory.search` or `calendar.create_event`.

## Capability Binding

The mapping between a KinOS capability and a concrete runtime tool, workflow or integration operation.

## Consent

An explicit authorization given by the owner or authorized subject for a defined action, sharing scope or external transfer.

## Integration

An adapter to an external or local system that implements one or more capabilities.

## Memory Item

A durable, structured, owned and governed unit of memory.

## Model

A language model used by a runtime. It is replaceable and does not own memory or policy.

## Policy

A rule evaluated by KinOS to allow, deny or require approval for an action.

## Policy Engine

The KinOS component responsible for evaluating policies before memory retrieval, runtime execution or capability use.

## Runtime

The technical system that runs agent sessions, model calls and tool execution. Hermes is the reference MVP runtime.

## Sphere

A governed unit of human representation. A Sphere can represent a person, family, team, company, association or institution.

## Sphere Agent

An agent representing a Sphere rather than an individual person.
